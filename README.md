# News Headline Web Scraper

A robust web scraper for extracting headlines from multiple news sources including BBC, CNN, Reuters, and Al Jazeera.

## Features

- 🌐 Scrapes headlines from multiple news sources
- 🔄 Automatic retry mechanism for failed requests
- 🎯 Random user-agent rotation to avoid detection
- 📊 Export to JSON and CSV formats
- 🚀 Easy to extend with new sources
- 📝 Comprehensive logging
